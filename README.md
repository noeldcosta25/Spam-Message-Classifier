# Spam Message Classifier (NLP)

This project detects whether a message is Spam or Not Spam using Natural Language Processing.

## Workflow
Text → TF-IDF Vectorization → Logistic Regression → Prediction

## Features
- Text preprocessing
- TF-IDF vectorization
- Machine learning classification
- User input prediction

## Example
Input: free money now
##
Output: Spam
